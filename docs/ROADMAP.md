# Roadmap

## Phase 1: Public Project Foundation

- Add open source documentation.
- Add license, contribution, security, and issue templates.
- Publish the repository under `aaindex`.

## Phase 2: Source Import

- Add the Samsung phone app source.
- Add the Mac telemetry agent source.
- Document local development and install steps.
- Add screenshots of the running dashboard.

## Phase 3: First Usable Release

- Provide a one-command Mac agent startup path.
- Provide Android install instructions.
- Add basic pairing or local access protection.
- Add release artifacts and version tags.

## Phase 4: Monitoring Features

- CPU usage and load.
- Memory pressure and swap.
- Battery state and charging status.
- Thermal state.
- Disk usage.
- Network throughput.
- Top processes by CPU and memory.

## Phase 5: Polish

- Dark dashboard optimized for phone display.
- Reconnect handling.
- Alerts for high temperature, battery, or memory pressure.
- Exportable logs for debugging.
