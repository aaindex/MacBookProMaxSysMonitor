# Code of Conduct

## Our Pledge

We want this project to be a welcoming, respectful place to build useful software together.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, and ask clarifying questions when something is unclear.
- Keep discussions focused on the project and the people using it.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without consent.
- Repeatedly derailing project discussions.
- Any behavior that makes participation unsafe or hostile.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct. Repeated or severe violations may lead to temporary or permanent bans from project spaces.
